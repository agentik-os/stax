# U-031: the zero-bleed rail · the quiet ⌘K · the responsive rules (RR-1..8)

TARGET: {{TARGET}}
REFERENCE: frameword/apps/crm-specimen (drills rail, .crumb-goto, crumb collapse)

## DETECT
- Drill/list hover fills or separators overhang the text rail (negative-margin
  gutter bleed, inset pseudo hairlines, hover width expansion) — glaring in dark.
- A labeled "GO TO ⌘K" pill lives in the topbar.
- No written responsive contract: crumbs overflow on deep paths, long titles
  push the crumbbar's right cluster out, tiers exist only as folklore.

## APPLY
1. ZERO-BLEED RAIL: rows sit ON the text rail — list margin 0, row padding
   12px 0, pseudo hairlines at left/right 0. The hover FILL, the separators and
   the text share EXACTLY the same bounds; hover turns the row's own hairline
   accent IN PLACE (no expansion choreography, no :has neighbor rules). Edge
   content BREATHES inside the fill: on hover the index nudges +10px and the
   arrow -10px by TRANSFORM only (zero layout shift) — text glued to a fill
   edge reads cramped, but padding would move the rail. Last row still paints
   no line (dedup law). Applies to .drills, connection rows, and list views.
2. THE QUIET ⌘K: kill the topbar pill. The palette's only chrome is a bare
   mono "⌘K" (11px, borderless, transparent) in the crumbbar, bottom-right,
   just LEFT of the GitHub icon — muted ink at rest, foreground + secondary
   fill on hover. Click opens the palette.
3. RESPONSIVE RULES RR-1..RR-8 (the spec's numbered contract): three tiers
   (≥900 docked / 640-900 overlay / <640 PushHost); ≤760 nav collapse; the
   crumbbar survives every width with ⌘K + GitHub + theme; crumbs
   middle-collapse to `home › … › parent › leaf` below 640 (the … focuses the
   nearest hidden panel; titles ellipsize at 96px); overlays clamp 88-92%;
   the document NEVER scrolls sideways; text degrades before it clips;
   compact refs keep their remove ×.

## VERIFY
- Hover a drill row in DARK: fill left/right == the rail (±1px), the accent
  hairline has zero inset offsets, a rest row's separator shares the fill's
  bounds; hovered index gap ≥ 8px, arrow gap ≥ 8px; a rest row's text offset
  from the row box is 0.
- `.tb-goto` matches nothing; `.crumb-goto` sits left of the GitHub icon and
  opens the palette on click.
- At 1280/980/744/580/375: `scrollWidth <= clientWidth`, the ⌘K chip and
  GitHub survive, and below 640 a path of 3+ shows the `…` crumb.
