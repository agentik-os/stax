# U-027: the UX Wave-1 baseline (fifteen grounded fixes)

TARGET: {{TARGET}}
REFERENCE: frameword/apps/crm-specimen (App.tsx, DataApp.tsx, styles.css @0.12.0)

## DETECT
Run the five-lens audit symptoms: focus walking off-screen, silent mouse pin,
Esc destroying selections, lying empty states, boxed chrome icons, off-accent
presence hues, destructive rhyming with primary, blinking toasts, pressless
drills, flat dark stacks, invisible creates, one-verb palettes.

## APPLY
1. Focus never leaves the viewport: [ / ] and the bar ‹ › scroll the focused
   panel into view (stage pad respected).
2. Every intent narrates on EVERY input: the mouse pin says what the keyboard
   pin says (one message, stax:say).
3. The Escape ladder gains a rung: a live table selection peels BEFORE any
   panel-level close.
4. Truthful empty states: "no rows YET" vs "no rows MATCH" branch on whether
   a search/filter is actually active.
5. Chrome recedes: topbar icon buttons lose their boxes (border transparent,
   hover tint only); presence dots use accent (filled=live), never a second hue.
6. Destructive foot actions rest in muted ink, red on approach only.
7. Toasts fade out (150ms) instead of blinking away; the drill gets an :active
   pressed state (deeper tint + arrow nudge).
8. Dark elevation parity: a denser directional panel shadow so the stack still
   reads as layers in dark.
9. Creates are visible and continuable: New row scrolls, FLASHES (700ms accent
   decay) and focuses the new row's first cell.
10. ⌘-Enter in the palette opens the result PINNED; "/" opens the focused
    panel's foot search; the keyboard map is a palette action too.
11. Root panel titles step up to serif 32 (weight 450, opsz 32); depth keeps 27.

## VERIFY
- Deep thread + [ ×2: the focused panel's rect sits inside the stage.
- Mouse-pin shows the toast; Esc with 2 rows selected clears the selection and
  the panel count is unchanged; a fresh table says "No rows yet".
- New row: activeElement is inside the LAST row. ⌘-Enter: the opened root is
  pinned/retained. Root title computes 32px, children 27px.
